# Loan Approval Prediction App

## 📊 Project Overview
This project predicts whether a loan will be approved based on user inputs like age, income, credit score, and loan amount.

## 🚀 Technologies Used
- Python
- Scikit-learn
- Streamlit
- NumPy

## ⚙️ Features
- Real-time prediction using Streamlit
- User-friendly interface
- Machine learning model integration

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
